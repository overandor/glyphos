# Membra ChatGPT Export Skill Pack

This pack converts the ChatGPT export folder into reusable agent skills.

The source folder contains ChatGPT export files about Gate.io trading systems, hedging bots, code appraisal, CSV inquiry, MEV experiments, Solana/NFT APIs, grant systems, workspace/project links, and value-chain audits.

The strongest extracted pattern is not a single bot. It is a workflow:

Conversation -> compressed artifact -> evidence -> receipt -> valuation -> underwriting -> product or protocol next step.

Included skills:

- chat_export_distiller
- provenance_to_collateral_auditor
- gateio_trading_system_auditor
- trading_code_appraiser
- compute_memory_continuity_underwriter
- receipt_ledger_designer
- github_underwriting_extractor
- proofbook_integration_planner
- solana_devnet_token_guard
- conversation_to_product_spec_compiler

Use these as drop-in SKILL.md folders for an agent environment that supports markdown skills.
